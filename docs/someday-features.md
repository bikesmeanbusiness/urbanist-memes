# Someday Features & Roadmap

This document tracks ideas and features that are out of scope for the MVP but are planned for the future.

## User Accounts & Personalization
- **Authentication**: Implement user login (e.g., Auth.js, Supabase).
- **Bookmarks**: Allow users to save their favorite memes.
- **Voting**: Up/down voting system to surface popular content.
- **Recommendation Engine**: Personalized feeds based on user activity.

## Content Creation & Remixing
- **Meme Generator**: Built-in tool to caption images.
- **Remixing**: Allow users to take an existing meme and edit the text or layout to create a variation.
- **Templates**: A library of urbanist-themed meme templates.

## Advanced Discovery
- **Semantic Search**: Vector-based search to find memes by concept.
- **Trends**: "Trending now" section based on view velocity.
